Big Data Mining and Business Analytics

Business-oriented insights derived from large-scale data using statistical and network analysis

This project was carried out as part of my Big Data Mining course and focuses on extracting business insights from large-scale datasets using data mining, network analysis, and machine learning techniques. My goal was to combine analytical accuracy with business reasoning — turning complex data into actionable insights about customer behavior, market structures, and inter-business connections.
